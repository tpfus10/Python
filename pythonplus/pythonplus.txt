[파이썬의 생성자]
__init__(): 클래스를 생성할 때 실행되는 생성자

[파이썬의 소멸자]
__del__(): 프로그램이 종료되면 자동으로 실행됨
->java의 garbage collection 역할

[파이썬 내장 메서드]
1.__len__(): 길이를 출력할 때 사용
2.__str__(): 객체를 String으로 출력할 때 사용
3.🟡__call__(): 클래스의 객체를 호출하게 해주는 메서드
->객체가 호출되면 __call__ 메소드가 실행됨
4.🟡__getitem__(): 클래스의 인덱스에 접근할 때 자동으로 호출되는 메서드 
5.__eq__(): 객체의 속성을 비교할 때 사용(T/F)
6.__feildname(): private feild 선언

[파이썬의 함수]
1.type 함수: 객체의 데이터 타입을 확인할 수 있는 함수
2.isinstance 함수: 객체가 해당 클래스의 객체인지 확인할 수 있는 함수

[파이썬의 클래스 변수]
클래스 내, 함수 밖에 선언되는 변수
함수들이 모두 공유함
함수 내에서는 '클래스명.변수명'으로 사용됨

[파이썬의 인스턴스 변수]
함수 안에서 선언되는 변수
'self.변수명'으로 사용됨

[파이썬의 클래스 메소드]
@classmethod(decorator)를 앞에 붙여줌
함수의 ()안에 self가 아닌 cls(class)를 넣어줌
'클래스명.클래스 메소드'로 호출함
